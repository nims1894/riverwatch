# CAB-020_CHANGELOG

## RiverWatch v1.0 RC3 — Boat Health Information Architecture

### Changed
- Boat Health now shows health-only metrics:
  - Trim Alignment
  - River Suitability
  - Role Coverage
  - Captain Discipline
- Renamed Diversification Integrity to Role Coverage.
- Boatyard now introduces a Boat Profile section:
  - Boat Archetype
  - Voyage Phase
  - Cost Basis
  - Boat Return
- Trim Deck remains responsible for allocation trim.

### Removed
- Boat Archetype, Voyage Phase, Cost Basis, and Boat Return from Dashboard Boat Health.
- Captain Discipline from Boatyard profile.

### Documentation
- Added Development Constitution.
- Added ADR-001 Health / Profile Separation.
