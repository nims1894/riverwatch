# RiverWatch v1.0 RC3 Release Note

## Theme

## Summary
RC3 separates health information from profile information.

Boat Health now answers: "How healthy is the boat?"

- Trim Alignment
- River Suitability
- Role Coverage
- Captain Discipline

## Boatyard
Boatyard now answers: "What kind of boat is this, and how is it trimmed?"

Sections:
- Boat Profile
- Trim Deck
